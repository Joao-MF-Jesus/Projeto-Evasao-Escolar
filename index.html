Análise e Monitoramento da Evasão Escolar no Brasil

Projeto G2 — Análise e Visualização de Dados com Python

Objetivo

Desenvolver um dashboard interativo para analisar indicadores relacionados à evasão escolar no Brasil, considerando fatores regionais, sociais e educacionais.

Tecnologias

Python
Pandas
Matplotlib
Seaborn
Streamlit
SQLAlchemy + SQLite

Principais Análises

  • Taxa média de evasão por região e UF
  • Evolução temporal da evasão escolar
  • Relação entre renda familiar, acesso à internet e evasão
  • Correlação estatística entre variáveis educacionais
  • Classificação dos registros por nível de risco
